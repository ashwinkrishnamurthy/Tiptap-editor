# Tiptap Editor

A rich text editor built with Tiptap.

## Getting Started

This project uses Tiptap, a headless wrapper around ProseMirror.

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Features

- Rich text editing
- Extensible with Tiptap extensions
- Modern and responsive design

## Tech Stack

- Tiptap
- JavaScript/TypeScript
- HTML/CSS
